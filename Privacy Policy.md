# BILITATO 隐私政策 (Privacy Policy)

最后更新：2026.3.14

BILITATO 是一款用于分析BiliBili视频内容的浏览器扩展。本隐私政策说明该扩展如何处理用户数据以及相关信息。

---

## 1. 数据收集 (Data Collection)

BILITATO **不会收集、存储或向开发者服务器传输任何个人数据**。

该扩展仅在用户浏览器本地运行，并只访问实现功能所必需的信息。

扩展可能访问的信息包括：

- 当前网页中的视频字幕内容
- 视频的基本页面信息（如视频标题）
- 用户自行配置的 AI API 设置

这些信息仅用于生成 AI 分析结果，例如：

- 视频总结
- 视频章节分段
- 广告识别
- 内容分析

所有处理均由用户浏览器或用户指定的 AI 服务完成。

---

## 2. API Key 与用户配置 (API Keys and User Configuration)

用户可以自行配置 AI 服务提供商，例如：

- OpenAI
- Gemini
- DeepSeek
- 智谱 AI
- ModelScope
- 其他兼容 API

相关说明：

- API Key **仅存储在用户浏览器本地**（使用 Chrome Storage）。
- API Key **不会发送给插件开发者**。
- API Key 仅用于向用户选择的 AI 服务发送请求。
- 开发者 **无法访问用户的 API Key**。

---

## 3. 本地数据存储 (Local Storage)

插件可能会在浏览器本地存储以下信息：

- 用户设置与偏好
- AI 服务提供商配置
- 已分析视频的缓存结果（如总结与分段）

这些数据 **仅保存在用户设备本地**，不会上传到任何服务器。

---

## 4. 网络请求 (Network Requests)

当用户使用 AI 分析功能时，插件可能会将视频字幕及相关上下文发送至用户配置的 AI API 服务。

这些请求：

- **直接从用户浏览器发送到 API 服务提供商**
- **不会经过开发者服务器**
- **不会被开发者存储或记录**

---

## 5. 第三方服务 (Third-Party Services)

如果用户使用第三方 AI API（例如 OpenAI、Gemini、DeepSeek 等），提交的数据将按照对应服务提供商的隐私政策进行处理。

用户在使用相关 API 前，应自行查阅对应服务的隐私政策。

---

## 6. 数据共享 (Data Sharing)

BILITATO **不会出售、共享或向第三方提供用户数据**。

---

## 7. 数据安全 (Security)

由于所有设置和缓存数据均保存在用户浏览器本地，用户始终对自己的数据拥有完全控制权。

用户可以通过以下方式删除数据：

- 卸载浏览器扩展
- 清除浏览器扩展存储

---

## 8. 隐私政策更新 (Policy Updates)

本隐私政策可能会不定期更新。如有更新，将在本页面更新最新版本。

---
